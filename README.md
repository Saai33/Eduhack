# EduHack - AI Powered Learning Platform

## Problem Statement
Students often struggle to access personalized learning resources, track progress, and collaborate effectively in one platform.

## Solution
EduHack is an AI-powered learning platform that provides study materials, quizzes, discussion forums, and performance tracking for students and teachers.

## Features
- Student Dashboard
- Teacher Dashboard
- Notes Management
- Quiz Creation & Attempt
- Discussion Forum
- Leaderboard
- Admin Panel

## Tech Stack
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Installation
1. Clone the repository
2. Import the database
3. Configure database connection
4. Run using XAMPP/WAMP
